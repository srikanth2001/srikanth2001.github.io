Projects

SimuLDPC:

A Django-based web application for the simulation of LDPC codes (a type of channel coding currently being tested in 5G standards) for three different types of channels. The project was done under the course, Independent Projects(EE2025). The decoding is done using graph-based algorithms like Belief Propagation and Gallagher-A algorithms. Tools: Python3(NumPY, MatplotLib), Django.

Voice Recognition Bot

Done as a part of the semester course (Intro to AI-ML EE1390) Implemented a raw neural network, using NumPy, for the voice recognition feature in an Arduino bot. The GUI can be used to transmit commands like forward, backward, turn left, turn right, and stop. The GUI also shows a prompt to the user to select whether the prediction is correct or not. Tools: Python3, Tkinter, HC-05 module, Arduino..

FM transmitter Circuit

Done as a part of semester course (communication systems EE1360) Assembled a FM transmitter circuit using resistors, capacitors, instructors, and trimmer capacitors. using trimmer capacitors, we can tune transmitter to a suitable frequency(eg:90.1MHz).
